# Vigilo Examples

This directory contains example projects demonstrating how to use Vigilo with different frameworks.

## Vue 3 + Vite Example

Located in `vue/`, this example shows how to use Vigilo with Vue 3 and the Composition API.

### Running the Example

```bash
cd vue
bun install
bun run dev
```

## React + Vite Example

Located in `react/`, this example shows how to use Vigilo with React.

### Running the Example

```bash
cd react
bun install
bun run dev
```

## Features Demonstrated

Both examples demonstrate:

- Basic Vigilo overlay setup
- Connecting tasks to page elements
- Freeroam mode for custom positioning
- Task status management
- Search functionality
- Settings panel
- Keyboard shortcuts
