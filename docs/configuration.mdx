# Configuration Options

Vigilo supports extensive customization through props and themes to match your application's design system.

## Basic Props

### Required Props

- **category**: `string` - The category identifier for organizing tasks
- **instanceId**: `string` (optional) - Unique identifier when multiple instances exist

### Optional Props

- **themeOverrides**: `ThemeOverrides` - Customize colors and styles
- **stylesOverrides**: `Styles` - Override CSS classes and component styles
- **colorMode**: `'auto' | 'light' | 'dark'` - Force color mode

## Theme Customization

### Colors

```tsx
const theme = {
  colors: {
    primary: '#3b82f6',      // Main accent color
    primaryDim: 'rgba(59, 130, 246, 0.1)',
    freeroam: '#8b5cf6',      // Connection mode color
    textMain: '#ffffff',        // Main text color
    textDim: '#9ca3af',         // Secondary text color
    bgPanel: '#000000',        // Panel background
    borderPanel: '#374151',     // Panel border
    bgOverlay: 'rgba(0, 0, 0, 0.3)'
  }
}
```

### Layout Classes

```tsx
const styles = {
  panel: 'fixed flex flex-col gap-2 rounded-lg border px-4 pb-4 pt-0 w-96 shadow-2xl transition-all backdrop-blur-md z-50',
  item: 'relative flex items-center gap-3 p-1.5 px-2 rounded transition-colors w-full',
  header: 'flex items-center justify-between gap-2 min-w-0 border-b pb-1',
  badge: 'shrink-0 rounded px-1.5 py-0.5 text-xs font-medium'
}
```

## Advanced Configuration

### Programmatic API

Create instances programmatically and control them via JavaScript:

```tsx
const { connect, disconnect, focus } = useVigiloConnection('backend-tasks')

// Connection management
connect('user-profile-task', document.querySelector('#profile'))
connect('api-endpoint-task', 'https://api.example.com/users')

// Task management
focus('backend-tasks', 0) // Focus on first task
```

### Instance Isolation

When using multiple Vigilo instances:

```tsx
// Each instance has isolated state
<Vigilo category="backend" instanceId="user-profile" />
<Vigilo category="frontend" instanceId="ui-components" />

// Keyboard shortcuts are scoped per instance
// Pressing 's' only opens the focused instance's settings
```

### Default Values

- **Maximum visible items**: 3 (adjustable via theme)
- **Undo window**: 8 seconds (adjustable via config)
- **Line opacity**: 0.7 (adjustable via prop)
- **Component opacity**: 0.9 (adjustable via prop)
